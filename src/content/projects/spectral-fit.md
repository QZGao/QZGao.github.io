`spectral_fit` provides a reproducible workflow for fitting pulsar radio spectra and comparing physically motivated models. It brings the model definitions, priors, inference, and diagnostic outputs into one inspectable package.

The project is part of a broader interest in making statistical machinery legible to the scientist using it: uncertainty should be visible, model assumptions should be recoverable, and results should be straightforward to reproduce.
