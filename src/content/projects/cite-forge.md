CiteForge turns a page's reference structure into an explicit working surface. Editors can inspect citation names and templates, find inconsistencies, and reorganise references without treating raw wikitext as an undifferentiated block.

It continues ideas first explored in the smaller reference maker hosted on this site, with a more deliberate data model and editing workflow.
